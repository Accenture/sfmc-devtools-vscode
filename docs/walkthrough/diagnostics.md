## Explore Diagnostics & Links

Once you have retrieved metadata, open any file to see:

- **Ctrl+Click navigation** on ContentBlockByKey() references, r__TYPE_key fields, and data extension names
- **Inline warnings** in the Problems panel for unresolvable references
- **Quick fixes** to retrieve missing items directly from the editor
- **Hover hints** for known SFMC system data views in SQL queries

Check the **SFMC DevTools** status bar item for cache status and feature toggles.
