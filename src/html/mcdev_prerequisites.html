Pre-Requisites Installation

  1. 1

    Install NodeJS

    1. ➡️ In a web browser, go to nodejs.org

    2. ➡️ Download the latest LTS version installer.

    3. ➡️ Run the installer on your operating system.

  2. 2

    Install GIT Command Line

    1. ➡️ In a web browser, go to git-scm.com/downloads

    2. ➡️ Under Downloads, click the icon for your operating system.

    3. ➡️ Run the installer on your operating system.
